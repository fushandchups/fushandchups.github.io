Making the website

Sat, Apr 25, 2020

First off I used Jekyll to create a static site. I chose hyde as theme because I liked the fixed side bar and minimal design. I used github pages to host the site. After playing with Jekyll a bit and having some trouble with ruby on my Mac, I swapped to using Hugo rather than Jekyll.

Creating a new site and initializing a git repository:

hugo new site fushandchups
cd fushandchups
git init

I use one respository for the site creation, which is private. The site itself is in its own repository that is public.

Cloning the theme repository. I used hyde which has been ported to Hugo by Mark Otto https://github.com/mdo.

cd themes
git clone https://github.com/spf13/hyde.git

Set the theme in the config.toml file to use the hyde theme

theme = "hyde"

Change the config.toml to fit your needs. Add some content:

hugo new posts/my-first-post.md

Preview the site with hugo server. The -D means include pages that are labelled ‘draft’.

hugo server -D

View the site at localhost:1313. Play with the look of the site and add whatever content you want. I added a colour to the theme for fushandchups. To build the site

hugo

This will create a public directory. This is the directory to push to your github repository username.github.io

cd public
git init
git add .
git commit -m 'latest version of site'
git remote add origin https://github.com/fushandchups/fushandchups.github.io.git
git push -u orign master

The wesbite is now hosted on github at fushandchups.github.io
